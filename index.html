How it works

Uses a queue (FIFO). Explores all neighbours at current depth before going deeper. Guarantees the shortest path in unweighted grids.

while queue:
  node = queue.popleft()
  for each neighbour:
    queue.append(nb)

Uses a stack (LIFO). Explores as deep as possible before backtracking. Finds a valid path but not necessarily shortest.

while stack:
  node = stack.pop()
  for each neighbour:
    stack.append(nb)
